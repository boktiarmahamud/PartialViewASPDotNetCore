# PartialViewAspDotNetCore

A simple ASP.NET Core project demonstrating the use of **Partial Views** to create reusable and modular UI components.

## Features

- Modular UI with Partial Views
- Easy to integrate into existing Razor Pages or MVC projects
- Demonstrates clean separation of layout and content
